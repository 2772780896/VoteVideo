# 测试数据生成说明

## 概述

为测试 VoteVideo 基本功能，使用 Prisma Seed 机制生成完整的 Mock 数据。

---

## 一、实现方式

使用 **Prisma Seed** 机制，通过脚本批量插入测试数据。

### 相关文件

| 文件 | 说明 |
|------|------|
| `prisma/seed.js` | 种子数据脚本，使用 Prisma Client 插入测试数据 |
| `package.json` | 添加 `prisma.seed` 配置，指向 seed 脚本 |

---

## 二、依赖安装

```bash
cd Node
npm install @faker-js/faker --save-dev
```

使用 `@faker-js/faker` 生成逼真的假数据（用户名、标题、正文等）。

## 四、图片和视频来源

### 图片（封面、头像等）

使用 **Picsum** 占位图服务：

```
https://picsum.photos/seed/{id}/width/height
```

- 固定种子（seed），同一 ID 每次返回同一张图
- 适合做测试数据，图片不会随机变化

示例：
- 用户头像：`https://picsum.photos/seed/testuser/200/200`
- 视频封面：`https://picsum.photos/seed/video0/640/360`
- 动态图片：`https://picsum.photos/seed/post0_1/400/300`

### 视频文件

使用 **w3schools** 的测试视频：

```
https://www.w3schools.com/html/mov_bbb.mp4
```

- 免费测试视频，短小可用
- 直接可访问，不需要本地存储

## 五、数据关联处理

使用 Prisma Transaction（`$transaction`）确保数据一致性，按以下顺序创建：

1. **清空现有数据**（按依赖倒序删除）
2. **创建标签**（获取 tid）
3. **创建用户**（获取 uid，密码统一加密为 123456）
4. **创建视频**（获取 vid，关联 1-3 个标签）
5. **创建文章**（获取 eid）
6. **创建动态**（获取 pid）
7. **创建评论**（先主评论，再子评论，更新 subCommentCount）
8. **创建关注关系**（更新用户的 followerCount 和 followingCount）
9. **创建收藏记录**
10. **创建历史记录**
11. **创建对话和消息**（更新 lastMessage 和 lastDate）
12. **创建通知**

## 八、Faker 数据生成示例

在 `seed.js` 中使用 `@faker-js/faker` 生成假数据：

```javascript
const { faker } = require('@faker-js/faker')

faker.person.fullName()     // 随机用户名
faker.internet.username()   // 随机用户名（无空格）
faker.lorem.sentence()      // 随机标题
faker.lorem.paragraph()     // 随机正文
faker.lorem.paragraphs(5, '\n\n')  // 多篇正文
faker.person.bio()          // 随机个人简介
faker.number.int({ min: 0, max: 100 })  // 随机数字
faker.datatype.boolean()    // 随机布尔值
faker.date.recent()         // 随机近期日期
faker.helpers.arrayElements(array, count)  // 从数组中随机选几个
```

## 九、注意事项

1. **密码加密**：使用 `bcrypt.hash('123456', 10)` 加密，与现有注册逻辑一致
2. **图片 URL**：使用 Picsum 占位符，固定 seed 保证图片不变
3. **视频 URL**：使用测试地址，实际项目需替换为真实视频文件
4. **清空数据**：seed 脚本开头会清空所有现有数据，谨慎运行
5. **子评论计数**：创建子评论后，需手动更新主评论的 `subCommentCount`
6. **关注计数**：创建关注关系后，需手动更新用户的 `followerCount` 和 `followingCount`

## 十、后续扩展

如需生成更多数据或调整数据类型，直接修改 `prisma/seed.js` 脚本：

- 增加数据数量：修改循环次数
- 新增数据类型：参考现有格式，按依赖顺序添加
- 调整关联关系：修改 `faker.helpers.arrayElements()` 的参数
