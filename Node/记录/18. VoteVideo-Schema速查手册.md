# VoteVideo 项目 Schema 速查手册

> 数据库：SQLite + Prisma ORM
> 更新时间：2026-06-14

本手册分为两部分：**A. 每张表的真实字段**（数据库里实际存的），**B. 表间关系总览**（谁连谁、用什么 include）。

---

## A. 每张表的真实字段（不含虚拟字段）

> **FK = 外键（Foreign Key）**，加粗的是指向其他表的字段，`?` 表示可选（可为空）。

| 表名 | 真实字段 | 说明 |
|------|---------|------|
| **User** | `uid` (PK, 自增), `username` (唯一), `password`, `profilePictureUrl?`, `info?`, `followerCount`, `followingCount`, `date` | 用户基础信息 |
| **Video** | `vid` (PK, 自增), `title`, `coverUrl?`, `videoUrl?`, `duration?`, `viewCount`, `commentCount`, `likeCount`, `favouriteCount`, `reshareCount`, `date`, **`uploader_uid`** (FK) | 视频 |
| **Essay** | `eid` (PK, 自增), `title`, `text`, `viewCount`, `commentCount`, `likeCount`, `favouriteCount`, `reshareCount`, `date`, **`uploader_uid`** (FK) | 文章 |
| **Post** | `pid` (PK, 自增), `title?`, `text?`, `pictureList?` (JSON), `videoList?` (JSON), `viewCount`, `commentCount`, `likeCount`, `reshareCount`, `favouriteCount`, `date`, **`uploader_uid`** (FK) | 动态 |
| **Tag** | `tid` (PK, 自增), `tagName` (唯一), `likeCount`, `favouriteCount`, `commentCount`, `viewCount` | 标签 |
| **Comment** | `cid` (PK, 自增), `text`, `type` (默认 'text'), `pictureList?` (JSON), `viewCount`, `likeCount`, `dislikeCount`, `favouriteCount`, `subCommentCount`, `date`, **`uploader_uid`** (FK), **`vid?`** (FK), **`eid?`** (FK), **`pid?`** (FK), **`tid?`** (FK), **`replyTo_cid?`** (FK) | 评论（可挂在 5 种媒体下） |
| **VideoTag** | **`vid`** (FK), **`tid`** (FK) | 视频-标签中间表（复合主键） |
| **UserFavourite** | **`uid`** (FK), `type`, `item_id`, `addDate` | 收藏（复合主键 uid+type+item_id） |
| **UserLike** | **`uid`** (FK), `type`, `item_id`, `addDate` | 点赞 |
| **UserHistory** | **`uid`** (FK), `type`, `item_id`, `addDate` | 浏览历史 |
| **UserReshare** | **`uid`** (FK), `type`, `item_id`, `addDate` | 转发 |
| **UserDislike** | **`uid`** (FK), `type`, `item_id`, `addDate` | 踩（仅 comment/tag） |
| **UserFollowing** | **`uid`** (FK), **`following_uid`** (FK) | 关注（复合主键） |
| **Dialogue** | `mid` (PK, 自增), **`participant1_uid`** (FK), **`participant2_uid`** (FK), `lastMessage?`, `lastDate?` | 对话 |
| **Message** | `id` (PK, 自增), **`mid`** (FK), **`sender_uid`** (FK), `text`, `date` | 对话中的消息 |
| **Notification** | `nid` (PK, 自增), `type`, **`from_uid`** (FK), **`to_uid`** (FK), `targetType`, `targetId`, `parentType?`, `parentId?`, `text`, `isRead`, `date` | 通知 |

## B. 表间关系总览

### User 表（中心表）

```
┌─────────────────────────────────────────────────────────────────────┐
│                         User（用户表）                                │
│  ┌───────┬────────────────────────────────────────────────────────┐ │
│  │ 指向   │  关系说明                                              │ │
│  ├───────┼────────────────────────────────────────────────────────┤ │
│  │ Video      │  用户上传的视频      include: { uploader: true }   │ │
│  │ Essay      │  用户上传的文章      include: { uploader: true }   │ │
│  │ Post       │  用户发布的动态      include: { uploader: true }   │ │
│  │ Comment    │  用户发布的评论      include: { uploader: true }   │ │
│  │ Tag        │  （无直接关系，通过 VideoTag 间接关联）             │ │
│  ├───────┼────────────────────────────────────────────────────────┤ │
│  │ UserFollowing │ 关注/粉丝   "Following"/"Follower" 命名关系     │ │
│  │ Dialogue      │ 对话        "DialogueParticipant1"/"2" 命名    │ │
│  │ Message       │ 消息        "SentMessages" 命名               │ │
│  │ Notification  │ 通知        "Sent/ReceivedNotifications" 命名 │ │
│  ├───────┼────────────────────────────────────────────────────────┤ │
│  │ UserFavourite │ 收藏    type 区分 video/essay/post             │ │
│  │ UserLike      │ 点赞    type 区分 video/essay/post/comment/tag │ │
│  │ UserHistory   │ 历史    type 区分 video/essay/post             │ │
│  │ UserReshare   │ 转发    type 区分 video/essay/post             │ │
│  │ UserDislike   │ 踩      type 区分 comment/tag                  │ │
│  └───────┴────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

### 五种媒体与评论的关系

```
Video ──uploader_uid──→ User         （一个视频属于一个用户）
Video ──comments──────→ Comment[]    （一个视频有多条评论）
Video ──videoTags─────→ VideoTag[]   （一个视频有多个标签，通过中间表）

Essay ──uploader_uid──→ User
Essay ──comments──────→ Comment[]

Post  ──uploader_uid──→ User
Post  ──comments──────→ Comment[]

Tag   ──videoTags─────→ VideoTag[]   （一个标签关联多个视频）
Tag   ──comments──────→ Comment[]

Comment ──vid───→ Video?            （评论属于一个视频，或…）
Comment ──eid───→ Essay?            （属于一篇文章，或…）
Comment ──pid───→ Post?             （属于一个动态，或…）
Comment ──tid───→ Tag?              （属于一个标签）
Comment ──replyTo_cid──→ Comment?   （回复的父评论）
```

### 对话与消息

```
Dialogue ──participant1_uid──→ User    （对话参与者1）
Dialogue ──participant2_uid──→ User    （对话参与者2）
Dialogue ──messages──────────→ Message[]
Message  ──mid───────────────→ Dialogue
Message  ──sender_uid────────→ User
```

### 通知

```
Notification ──from_uid──→ User    （谁发的）
Notification ──to_uid────→ User    （发给谁）
```

## C. Service 层常用 include 速查

> **写 service 时不需要记全部关系**：只关心当前表的外键字段，用 include 把需要的关联数据带出来即可。

| Service | 常用 include | 说明 |
|---------|--------------|------|
| videoService | `include: { uploader: true }` | 查视频带上传者信息 |
| videoService | `include: { videoTags: { include: { tag: true } } }` | 查视频带标签列表 |
| essayService | `include: { uploader: true }` | 查文章带上传者信息 |
| postService | `include: { uploader: true }` | 查动态带上传者信息 |
| tagService | `include: { videoTags: { include: { video: true } } }` | 查标签带关联视频 |
| commentService | `include: { uploader: true, parentComment: true }` | 查评论带评论者和父评论 |
| interactService | `where: { uid, type: 'video' }` | 查用户收藏/点赞/历史（按 type 过滤） |
| messageService | `include: { sender: true, dialogue: true }` | 查消息带发送者和对话信息 |
