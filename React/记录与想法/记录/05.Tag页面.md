# Tag 页面 — 开发记录

> Tag 标签详情页：套用 Essay/Post 同模板，标签名+统计数据+InteractionBar（showReshare=false）+评论区。

## 关键词
Tag, 标签, InteractionBar, showReshare

## 涉及文件
src/pages/Tag.jsx

---

## 一、页面职责

标签详情页，展示标签名称、浏览量/评论数等统计数据，支持点赞/收藏（不支持转发）和评论区。

接口：`fetchItem('tag', tid)` 获取详情。字段定义见 [前端接口分析.md](./13.%20VoteVideo-前端接口分析.md) 第二节。

---

## 二、开发历程

与 Essay/Post 同模板开发：

```
标签名
浏览量 / 评论数
InteractionBar（showReshare={false}，标签不支持转发）
评论区 DataList
```

Mock 中 tag 有 `likeCount`、`favouriteCount` 字段，直接对接 InteractionBar。

---

## 三、架构决策

### 3.1 标签不支持转发

标签是内容分类标记，不是用户发布的内容，转发语义不通。InteractionBar 通过 `showReshare={false}` 隐藏转发按钮。

---

## 四、组件结构

```
Tag.jsx
├── TopMenuApp
├── 标签信息区（名称 + 统计）
├── InteractionBar（mediaType="tag"，showReshare=false）
└── DataList（评论区）
```
