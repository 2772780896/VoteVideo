# Essay 页面 — 开发记录

> Essay 文章详情页：从 antd 全量迁移到 Tailwind，引入 useData 四状态解构（loading/error/empty/content），接入通用 InteractionBar 和 DataList 直调评论。

## 关键词
Essay, 四状态, useData, Tailwind, InteractionBar, DataList, antd迁移

## 涉及文件
src/pages/Essay.jsx, src/components/common/InteractionBar.jsx, src/hooks/useData.js

---

## 一、页面职责

文章详情页，展示文章标题、作者、正文，支持点赞/收藏交互和评论区。

接口：`fetchItem('essay', eid)` 获取详情。字段定义见 [API与数据格式.md](./11.API与数据格式.md) 第四节。

---

## 二、开发历程

### 2.1 antd → Tailwind 全量迁移

| antd | Tailwind 替代 |
|------|--------------|
| `<Row>` | `<div>` |
| `<Col span={16} offset={4}>` | `<main className="max-w-3xl mx-auto px-4 py-6">` |
| `<Flex gap="middle">` | `<div className="flex items-center gap-3">` |
| `<Avatar>` | `<div className="w-12 h-12 rounded-full bg-gray-300 ...">` |

### 2.2 useData 四状态解构

```js
const { data, loading, error, refresh } = useData(fetchItem, 'essay', eid)
```

### 2.3 渲染四态分支

```
loading?  ──yes──→ 骨架屏
  no
error?    ──yes──→ code===404 ? "文章不存在" : "加载失败" + 重试
  no
essay?    ──no───→ "文章不存在"（空态）
  yes
正常渲染内容
```

### 2.4 接入通用组件

- 评论区：`<DataList>` 直调 `fetchItemList('comment', ...)`，废弃 CommentFlexApp 包装层
- 交互按钮：`<InteractionBar mediaType="essay" item={essay} />`

> InteractionBar 通用设计见 [通用组件与Hook.md](./通用组件与Hook.md) 第三节。
