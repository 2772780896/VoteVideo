## 一、页面职责

用户公开页（/user/:uid），展示指定用户的头像/用户名/粉丝关注统计，以及该用户发布的视频/动态/文章列表。

接口：`fetchItem('user', uid)` 获取用户资料。字段定义见 [前端接口分析.md](./13.%20VoteVideo-前端接口分析.md) 第一、二节。

## 二、开发历程

### 2.1 antd → Tailwind + 四状态

| 纬度 | 旧 | 新 |
|------|-----|-----|
| API | `getUserPublicProfile(uid)` | `fetchItem('user', uid)` |
| 用户信息 | antd Row/Col/Avatar | Tailwind flex + img + 统计 |
| Tabs | antd Segmented + Tabs | Tailwind button tabs |
| 三态 | 无 | 骨架屏 / 错误+重试 / 空态 |

### 2.2 子组件废弃

UserVideoFlex / UserPostFlex / UserEssayFlex 三个文件 99% 相同：收数组 → map 卡片 → antd Flex。直接换成 Tailwind 网格 + inline map。

### 2.3 关注按钮

```jsx
const handleFollow = async () => {
  const next = !isFollowing
  try {
    await interact('user', next ? 'follow' : 'unfollow', user.uid)
    setIsFollowing(next)
  } catch {}
}
```

### 2.4 VideoCard stopPropagation 模式

整张卡链到 `/video/:vid`，用户名区域单独链到 `/user/:uid`：

```jsx
<Link to={`/video/${video.vid}`}>
  <Link to={`/user/${video.uploader?.uid}`} onClick={(e) => e.stopPropagation()}>
    {video.uploader?.userName}
  </Link>
</Link>
```

stopPropagation 保留了外层 hover 动画和 group-hover CSS 效果，比拆散式方案（去掉外层 Link）更好。
